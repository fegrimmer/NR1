# NR1 Inteligente

Projeto de software SaaS para gestão de riscos psicossociais conforme a NR-01,
com foco em comunicação, adesão dos colaboradores e governança técnica.

Este repositório contém a **documentação viva** do projeto,
utilizada como fonte única de verdade para definição funcional,
arquitetural e técnica.

---

## Status

Documentação inicial concluída (v0.1.0), em validação.

Nenhuma decisão de stack ou implementação foi tomada até este ponto.

---

## Escopo Atual

NR-01 – Gestão de Riscos Psicossociais, incluindo:
- análise de maturidade organizacional
- avaliação psicossocial
- engenharia de segurança
- gestão contínua e monitoramento de riscos
- integração com sistemas de SST (ex: SOC)
